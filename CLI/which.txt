[X] which Command List

[\] which command
[/] Show full path of executable command

[\] which ls bash python
[/] Show paths for multiple commands

[\] which -a command
[/] Show all matching paths (not just first)

[\] which -a python
[/] Show all Python executables in PATH (multiple versions)

[\] which -s command
[/] Silent: return exit code only (no output)

[\] which --help
[/] Show help menu

[\] which --version
[/] Show which version

[\] which -a ls
[/] Show all ls paths (e.g., /bin/ls, /usr/bin/ls)

[\] if which python >/dev/null; then echo "Found"; fi
[/] Check if command exists (supress output)

[\] which -a --skip-alias command
[/] Skip aliases and show real executable

[\] which --read-alias command
[/] Read aliases from shell (if supported)

[\] which --show-dot command
[/] Include dot (.) directories in search

[\] which --show-tilde command
[/] Show tilde (~) for home directory when applicable

[\] which --tty-only command
[/] Stop processing options if not on TTY

[\] type command
[/] Alternative: show if command is alias, builtin, or executable

[\] type -a command
[/] Show all locations (aliases, builtins, binaries)

[\] command -v command
[/] POSIX alternative: show path or command description

[\] whereis command
[/] Show binary, source, and man page locations

[\] whereis -b command
[/] Show only binary locations

[\] which -a --skip-functions command
[/] Skip shell functions in search

[\] which --skip-dot command
[/] Skip dot directories in search

[\] which --skip-tilde command
[/] Skip tilde directories in search

[\] which --skip-alias command
[/] Skip aliases (same as --skip-alias)
