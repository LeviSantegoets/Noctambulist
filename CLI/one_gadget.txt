[X] one_gadget Command List

[] one_gadget libc_file
[] Find one-shot RCE gadgets in a libc binary

[] one_gadget libc_file -l
[] Show gadgets with more detailed constraints

[] one_gadget libc_file -r
[] Show gadgets with register constraints

[] one_gadget libc_file --level n
[] Set output detail level (higher = more constraints shown)

[] one_gadget libc_file --raw
[] Output only gadget offsets (raw format)

[] one_gadget libc_file --base address
[] Calculate gadget addresses with a base address

[] one_gadget libc_file --near function
[] Find gadgets near a specific function

[] one_gadget libc_file --script
[] Output results in a script-friendly format

[] one_gadget --help
[] Show help menu

[] one_gadget --version
[] Show one_gadget version
