[X] ZGREP Command List

[\] zgrep --help
[/] Show help menu

[\] zgrep --version
[/] Show zgrep version

[\] zgrep "<pattern>" <file.gz>
[/] Search a gzip-compressed file using regular expressions

[\] zgrep "<pattern>" <file1.gz> <file2.gz>
[/] Search multiple compressed files

[\] zgrep -i "<pattern>" <file.gz>
[/] Ignore case while searching

[\] zgrep -n "<pattern>" <file.gz>
[/] Show matching line numbers

[\] zgrep -c "<pattern>" <file.gz>
[/] Count matching lines

[\] zgrep -l "<pattern>" *.gz
[/] List compressed files containing matches

[\] zgrep -v "<pattern>" <file.gz>
[/] Show non-matching lines

[\] zgrep -w "<pattern>" <file.gz>
[/] Match whole words only

[\] zgrep -o "<pattern>" <file.gz>
[/] Print only matched text

[\] zgrep -A 3 "<pattern>" <file.gz>
[/] Show 3 lines after each match

[\] zgrep -B 3 "<pattern>" <file.gz>
[/] Show 3 lines before each match

[\] zgrep -C 3 "<pattern>" <file.gz>
[/] Show 3 lines of surrounding context

[\] zgrep -r "<pattern>" <directory>
[/] Recursively search compressed files in directory (if supported)

[\] zgrep --help-all
[/] Show advanced options
