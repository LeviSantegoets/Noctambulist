[X] ltrace-ng Command List

[\] ltrace-ng -h
[/] Show help menu

[\] ltrace-ng <binary>
[/] Trace library calls of a binary

[\] ltrace-ng -p <pid>
[/] Attach to a running process by PID

[\] ltrace-ng -f <binary>
[/] Follow child processes

[\] ltrace-ng -S
[/] Show system calls along with library calls

[\] ltrace-ng -o <file>
[/] Save output to a file

[\] ltrace-ng -e <function>
[/] Trace only specific function calls

[\] ltrace-ng -i <interval>
[/] Set polling interval for tracing

[\] ltrace-ng -v
[/] Enable verbose output

[\] ltrace-ng --version
[/] Show ltrace-ng version
