[X] Ripgrep (rg) Full Command List

[\] rg pattern
[/] Search for pattern recursively in current directory

[\] rg pattern file.txt
[/] Search pattern in a specific file

[\] rg -i pattern
[/] Case-insensitive search

[\] rg -v pattern
[/] Invert match (show lines that do not match)

[\] rg -w pattern
[/] Match whole words only

[\] rg -x pattern
[/] Match whole line exactly

[\] rg -n pattern
[/] Show line numbers with matches

[\] rg -H pattern
[/] Show filename for each match

[\] rg -h pattern
[/] Suppress filenames in output

[\] rg -C 3 pattern
[/] Show 3 lines of context before and after match

[\] rg -A 3 pattern
[/] Show 3 lines after match

[\] rg -B 3 pattern
[/] Show 3 lines before match

[\] rg --color=always pattern
[/] Force colored output

[\] rg --color=auto pattern
[/] Auto color matches

[\] rg --color=never pattern
[/] Disable colored output

[\] rg -l pattern
[/] List files containing the pattern

[\] rg -L pattern
[/] List files not containing the pattern

[\] rg --files
[/] List all files that would be searched

[\] rg --type rust pattern

[\] rg --max-depth 2 pattern

[\] rg --hidden pattern
[/] Include hidden files

[\] rg --follow pattern
[/] Follow symbolic links

[\] rg --threads 4 pattern
[/] Use 4 threads for searching

[\] rg --stats pattern

[\] rg --no-ignore pattern

[\] rg --no-ignore-global pattern

[\] rg --no-ignore-vcs pattern

[\] rg --multiline pattern

[\] rg --word-regexp pattern

[\] rg --max-columns 200 pattern
