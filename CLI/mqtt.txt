[\] mqtt --help
[/] Show help menu

[/] Show MQTT-CLI version

[/] Subscribe to a topic

[\] mqtt pub -t <topic> -m "<message>" -h <broker_host>
[/] Publish a message to a topic

[\] mqtt pub -t <topic> -f <file> -h <broker_host>
[/] Publish messages from a file
