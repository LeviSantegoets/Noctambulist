# Proxmox VE Command List
# Format: [\] command [/] description [X] menutitlename

[X] Proxmox VE Command List

[\] pveversion
[/] Show Proxmox VE version

[\] pveversion -v
[/] Show detailed Proxmox package versions

[\] systemctl status pvedaemon
[/] Check PVE daemon status

[\] systemctl restart pvedaemon
[/] Restart PVE daemon

[\] systemctl restart pveproxy
[/] Restart web interface

[\] systemctl restart pvestatd
[/] Restart Proxmox statistics service

[\] systemctl restart pve-cluster
[/] Restart cluster service

[\] systemctl restart corosync
[/] Restart Corosync (cluster communication)

[\] pvecm status
[/] Show cluster status

[\] pvecm nodes
[/] List cluster nodes

[\] pvecm add <IP>
[/] Join a node to a cluster

[\] pvecm delnode <node>
[/] Remove a node from cluster

[\] pvecm expected <number>
[/] Set expected quorum votes

[\] qm list
[/] List all virtual machines

[\] qm start <VMID>
[/] Start a VM

[\] qm stop <VMID>
[/] Stop a VM

[\] qm shutdown <VMID>
[/] Gracefully shut down a VM

[\] qm reboot <VMID>
[/] Reboot a VM

[\] qm suspend <VMID>
[/] Suspend a VM

[\] qm resume <VMID>
[/] Resume a VM

[\] qm create <VMID> --name testvm --memory 2048
[/] Create a new VM

[\] qm destroy <VMID>
[/] Delete a VM

[\] qm config <VMID>
[/] Show VM configuration

[\] qm set <VMID> --memory 4096
[/] Change VM memory

[\] qm set <VMID> --cores 4
[/] Change VM CPU cores

[\] qm set <VMID> --net0 virtio,bridge=vmbr0
[/] Add or modify network interface

[\] qm set <VMID> --scsi0 local-lvm:32
[/] Add a disk to VM

[\] qm resize <VMID> scsi0 +20G
[/] Resize VM disk

[\] qm migrate <VMID> <target-node>
[/] Migrate VM to another node

[\] qm snapshot <VMID> snap1
[/] Create a snapshot

[\] qm listsnapshot <VMID>
[/] List snapshots

[\] qm delsnapshot <VMID> snap1
[/] Delete a snapshot

[\] pct destroy <CTID>

[\] pct config <CTID>

[\] pct set <CTID> -memory 2048

[\] pct set <CTID> -cores 2
