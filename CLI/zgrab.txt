[X] ZGRAB Command List

[\] zgrab --help
[/] Show help menu

[\] zgrab --version
[/] Show zgrab version

[\] zgrab <module>
[/] Run a scan using a specific module (e.g., http, tls, ssh)

[\] zgrab http --port <port> <target_subnet>
[/] Scan HTTP service on target hosts

[\] zgrab tls --port <port> <target_subnet>
[/] Scan TLS/SSL services and extract certificates

[\] zgrab ssh --port <port> <target_subnet>
[/] Scan SSH services and gather banner info

[\] zgrab smtp --port <port> <target_subnet>
[/] Scan SMTP services and capture responses

[\] zgrab ftp --port <port> <target_subnet>
[/] Scan FTP services and extract banners

[\] zgrab -o <output_file>
[/] Save scan results to file

[\] zgrab -f <input_file>
[/] Read targets from input file

[\] zgrab -t <timeout>
[/] Set per-connection timeout

[\] zgrab -r <rate>
[/] Set scan rate (requests per second)

[\] zgrab --output-format=json
[/] Output results in JSON format

[\] zgrab --senders <count>
[/] Set number of concurrent workers

[\] zgrab --retry <count>
[/] Set number of retries for failed connections

[\] zgrab --quiet
[/] Suppress non-essential output

[\] zgrab --help-all
[/] Show advanced options
