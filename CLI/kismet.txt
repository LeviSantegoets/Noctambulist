[X] Kismet Command List

[\] sudo apt install kismet
[/] Install Kismet on Debian‑based systems

[\] sudo systemctl start kismet
[/] Start Kismet service

[\] sudo systemctl stop kismet
[/] Stop Kismet service

[\] sudo systemctl enable kismet
[/] Enable Kismet at boot

[\] sudo systemctl status kismet
[/] Check service status

[\] kismet
[/] Start Kismet server + UI (default)

[\] kismet -c wlan0
[/] Start Kismet using interface wlan0

[\] kismet -c wlan0mon
[/] Start Kismet using monitor‑mode interface

[\] kismet_server
[/] Start server only (headless mode)

[\] kismet_client
[/] Start client UI only

[\] kismet --no-ncurses
[/] Run without terminal UI (server only)

[\] sudo kismet_cap_linux_wifi --list-interfaces
[/] List available wireless interfaces

[\] sudo kismet_cap_linux_wifi --capture-source=wlan0
[/] Start capture on wlan0

[\] sudo kismet_cap_linux_wifi --channel=6
[/] Set channel manually

[\] sudo kismet_cap_linux_wifi --channels=1,6,11
[/] Hop between channels

[\] sudo kismet_cap_linux_wifi --monitor
[/] Force monitor mode on interface

[\] sudo nano /etc/kismet/kismet.conf
[/] Edit main Kismet configuration

[\] sudo nano /etc/kismet/kismet_site.conf
[/] Edit site‑specific overrides

[\] sudo nano ~/.kismet/kismet.conf
[/] User‑level configuration

[\] source=wlan0
[/] Add capture source

[\] source=wlan0:name=wifi0
[/] Add source with custom name

[\] log_prefix=logs/
[/] Set log directory

[\] log_types=pcapng,netxml,nettxt
[/] Enable specific log formats

[\] channel_hop=true
[/] Enable channel hopping

[\] channel_hop_speed=5
[/] Set hop speed (seconds)

[\] kismetdb_stats log.kismet

[\] kismetdb_dump log.kismet

[\] kismetdb_dump --records

[\] kismetdb_to_pcap log.kismet output.pcap

[\] kismetdb_to_wiglecsv log.kismet output.csv
