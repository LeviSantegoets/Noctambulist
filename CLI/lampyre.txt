# Lampyre Command List
# Format: [\] command [/] description [X] menutitlename

[X] Lampyre Command List

# -------------------------
# INSTALLATION & STARTUP
# -------------------------

[\] lampyre
[/] Launch Lampyre application

[\] lampyre --safe-mode
[/] Start Lampyre in safe mode

[\] lampyre --reset-settings
[/] Reset Lampyre configuration

[\] lampyre --log-level debug
[/] Start Lampyre with debug logging


# -------------------------
# PROJECT MANAGEMENT
# -------------------------

[\] New Project
[/] Create a new investigation project

[\] Open Project
[/] Open an existing Lampyre project

[\] Save Project
[/] Save the current project

[\] Export Project
[/] Export investigation results

[\] Import Data
[/] Import external data into project

[\] Company Investigation
