[X] ZFGREP Command List

[\] zfgrep --help
[/] Show help menu

[\] zfgrep --version
[/] Show zfgrep version

[\] zfgrep "<text>" <file.gz>
[/] Search a gzip-compressed file for fixed text (no regex)

[\] zfgrep "<text>" <file1.gz> <file2.gz>
[/] Search multiple compressed files

[\] zfgrep -i "<text>" <file.gz>
[/] Ignore case while searching

[\] zfgrep -n "<text>" <file.gz>
[/] Show matching line numbers

[\] zfgrep -c "<text>" <file.gz>
[/] Count matching lines

[\] zfgrep -l "<text>" *.gz
[/] List compressed files containing matches

[\] zfgrep -v "<text>" <file.gz>
[/] Show non-matching lines

[\] zfgrep -w "<text>" <file.gz>
[/] Match whole words only

[\] zfgrep -o "<text>" <file.gz>
[/] Print only matched text

[\] zfgrep -A 3 "<text>" <file.gz>
[/] Show 3 lines after each match

[\] zfgrep -B 3 "<text>" <file.gz>
[/] Show 3 lines before each match

[\] zfgrep -C 3 "<text>" <file.gz>
[/] Show 3 lines of surrounding context

[\] zfgrep -r "<text>" <directory>
[/] Recursively search compressed files in directory (if supported)

[\] zfgrep --help-all
[/] Show advanced options
