[X] usql Command List

[] usql --help
[] Show help menu

[] usql --version
[] Show usql version

[] usql driverusernamepassword@hostportdatabase
[] Connect to a database using a specific driver (PostgreSQL, MySQL, SQLite, etc.)

[] q
[] Quit usql

[] c connection_string
[] Connect to another database

[] d
[] List all tables in current database

[] d table_name
[] Describe a specific table

[] dt
[] List all tables only

[] df
[] List all functions

[] dv
[] List all views

[] l
[] List all databases

[] h SQL_command
[] Show syntax help for SQL command

[] ! shell_command
[] Run shell command without leaving usql

[] e
[] Open query editor

[] x
[] Toggle expanded output

[] timing
[] Toggle query execution timing

[] watch seconds query
[] Repeatedly run a query every N seconds

[] set variable value
[] Set a session variable

[] unset variable
[] Unset a session variable

[] source file.sql
[] Execute SQL file

[] help
