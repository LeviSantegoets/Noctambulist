# Cisco IOS Command List

# EXEC & Privileged Mode

[\] enable
[/] Enter privileged EXEC mode

[\] disable
[/] Exit privileged EXEC mode

[\] exit
[/] Exit current mode

[\] logout
[/] Log out of the session

# File & Configuration Management

[\] copy running-config startup-config
[/] Save running config

[\] write memory
[/] Save running config (same as copy run start)

[\] write erase
[/] Erase startup config

[\] erase startup-config
[/] Delete startup config

[\] erase flash:
[/] Erase flash storage

[\] copy startup-config running-config
[/] Merge startup config into running config

[\] copy tftp: running-config
[/] Restore config from TFTP

[\] copy running-config tftp:
[/] Backup config to TFTP

[\] reload
[/] Reboot device
