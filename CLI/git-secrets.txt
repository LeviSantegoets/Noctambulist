[X] git-secrets Command List

[\] git secrets --install
[/] Install git-secrets hooks in the current repository

[\] git secrets --register-aws
[/] Register AWS patterns (access keys, secret keys)

[\] git secrets --add <pattern>
[/] Add a custom regex pattern to scan for secrets

[\] git secrets --scan
[/] Scan the current repository for secrets

[\] git secrets --scan-history
[/] Scan the entire Git history for secrets

[\] git secrets --list
[/] List all registered patterns

[\] git secrets --edit
[/] Edit patterns manually

[\] git secrets --uninstall
[/] Remove git-secrets hooks from the repository

[\] git secrets --pre-commit
[/] Run pre-commit check manually

[\] git secrets --version
[/] Show git-secrets version

[\] git secrets --help
[/] Show help menu
