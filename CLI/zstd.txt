[X] ZSTD Command List

[\] zstd --help
[/] Show help menu

[\] zstd --version
[/] Show zstd version

[\] zstd <file>
[/] Compress a file using default compression level

[\] zstd -d <file.zst>
[/] Decompress a .zst file

[\] zstd -c <file> > <file.zst>
[/] Compress data and write to stdout

[\] zstd -o <output.zst> <file>
[/] Specify output file name

[\] zstd -f <file>
[/] Force overwrite existing output file

[\] zstd -k <file>
[/] Keep original file after compression

[\] zstd -1 <file>
[/] Fast compression (lowest compression level)

[\] zstd -9 <file>
[/] High compression (slower, better ratio)

[\] zstd --ultra <file>
[/] Enable ultra compression levels

[\] zstd -T0 <file>
[/] Use all available CPU threads

[\] zstd -t <file.zst>
[/] Test integrity of compressed file

[\] zstd --long <file>
[/] Enable long distance matching mode

[\] zstd --quiet <file>
[/] Suppress output messages

[\] zstd --verbose <file>
[/] Show detailed compression info

[\] zstd --help-all
[/] Show advanced options
